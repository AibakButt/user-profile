# Optimzed User Profile Search

This repository contains a User Management System developed using React, Apollo Client, GraphQL, Express, Node.js, and MongoDB. The system offers features to manage user records efficiently through a user-friendly interface.

## Features

- Display a list of users in a tabular format.
- Sort users based on different columns (Name, Email, Phone, LinkedIn, Company) in both ascending and descending order.
- Search functionality to find users based on specified queries.
- Pagination for easy navigation through user records.
- Real-time loading indicator for a smooth user experience.

## Getting Started

### Installation

1. Clone the repository:

  ```bash
  git clone https://github.com/AibakButt/user-profile
  cd user-profile
  cd backend
  npm install

  cd ../frontend
  npm install
  ```

To strat frontend: ``` npm run dev ```

To strat backend: ``` npm run start:dev ```

**NOTE:** The mongodb already has 1 million user records, if you want to add more records.
Run command: ``` npm run seed```

Access the application in your web browser at http://localhost:3000.
